ec2
====

Stuffs to Manage My Amazon EC2 machine


これはなに？
------------

自分の持ってる Amazon EC2 のインスタンスを管理するためのレポジトリです。

鍵があれば、 `ssh ubuntu@10sr.mydns.jp` でログインできます。
